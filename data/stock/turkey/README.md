# Turkish Stock Source Documents

This folder stores source documents for Turkish listed companies.

- Company folders use lowercase BIST ticker keys, such as `mgros` and `tuprs`.
- PDFs should be stored under `reports/`, `investor-presentations/`, or `financial-statements/`.
- Every PDF should be registered in the relevant company `manifest.json` file.
- File names should use lowercase kebab-case.
- This structure is intended for future investment analytics and chatbot/RAG features.
