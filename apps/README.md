# Applications

## Purpose

Future home for user-facing applications such as the public website, authenticated dashboard, and admin surfaces.

## What belongs here

Application shells, route entry points, app-specific composition, and app-local documentation after explicit migration PRs.

## What does not belong here yet

Existing public HTML/CSS/JS routes, dashboard runtime files, package scripts, Firebase/auth behavior, or compatibility wrappers until a route-safe migration PR moves them.

## Migration status

Phase 2 skeleton only; no runtime files have been moved here yet.
