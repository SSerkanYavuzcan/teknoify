# Shared Packages

## Purpose

Future home for reusable shared modules such as UI, auth, config, charts, data-access, and utilities.

## What belongs here

Cross-app modules with stable APIs, package-level documentation, and migrated reusable code after explicit PRs.

## What does not belong here yet

Application routes, domain-specific business rules, Firebase config changes, or package script changes yet.

## Migration status

Phase 2 skeleton only; no runtime files have been moved here yet.
