# Archive

## Purpose

Future home for files intentionally retired from the active architecture after reviewed migration PRs.

## What belongs here

Date-and-reason grouped archive folders such as `_archive/2026-06-pre-domain-refactor/`; each archive folder must include a README explaining why the files were archived, when, and what replaced them.

## What does not belong here yet

Active runtime files, temporary scratch files, unreviewed deletions, or files moved without a migration PR.

## Migration status

Phase 2 skeleton only; no runtime files have been moved here yet.
