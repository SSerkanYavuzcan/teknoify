    API Preview

    Demo Lab ileride API tabanlı otomasyon çalıştırma, webhook ve entegrasyon akışları için genişletilebilir.

    POST/v1/demos/run
    GET/v1/demos/{id}/result

    API Preview
    const response = await fetch("https://api.teknoify.com/v1/demos/run", {
  method: "POST",
  headers: {
    Authorization: "Bearer tk_demo_key",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    demo: "catalog-health-check",
    mode: "sandbox"
  })
});

const result = await response.json();
console.log(result);
